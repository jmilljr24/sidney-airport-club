# Adding Meeting Minutes

## Quick Steps

### 1. Navigate to the Minutes Folder

Go to the [`_minutes`](https://github.com/justinliucs/sidney-airport-club/tree/main/_minutes) folder in the repository.

### 2. Upload a File

Click the **"Add file"** button, then select **"Upload files"**.

### 3. Choose Your File

Drag and drop or select your `.docx` file. The file must be named using the date format `YYYY-MM-DD.docx` (e.g., `2026-02-16.docx`).

### 4. Commit Changes

1. Click **"Commit changes..."**
2. Select **"Create a new branch"**
3. Name your branch (e.g., `minutes-2026-02-16`)
4. Click **"Propose changes"**

### 5. Open a Pull Request

GitHub will redirect you to open a Pull Request. Add a description and click **"Create pull request"**.

### 6. Merge the Pull Request

Once you're ready, click **"Merge pull request"** and confirm. The GitHub Action will automatically:

- Process the minutes file
- Generate the minutes page
- Update the website

## That's It

After merging, the new minutes will appear on the website within a few minutes.
